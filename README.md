# Last Breath - A Survival Game

Fight them off. Don't get tired. Survive.

This is a wave-based zombie survival game where you fight off zombies as long as you can. Attacking takes stamina, regenerate by standing still.

## How to play

Use A/D or left and right arrows to move. W to jump, and Space to attack. You have 5 stamina points, and each attack takes away one. Stand still to regenerate stamina.

### Download the project yourself

*Note: You must have Unity version 6000.2.10f1 downloaded*

#### **Download Via .zip**

1. Go to the repository's main page.
   
	1a. Click the green "Code" button.

	1b. Click "Download ZIP"

2. In your downloads folder, locate the **.zip** file you just downloaded.
   
	2a. Right Click > "Extract All..."

	2b. Choose the folder where to extract the **.zip** *(Note: It's best to pick the folder containing your other Unity projects).*

3. Open Unity Hub and go to the "Projects" tab on the left-hand side of the Hub window.

   *(Note: Make sure you open the project using the right Unity version!)*

	3a. In the top-right corner, click "Add" > "Add project from disk"

	3b. Select the folder containing the project. Unity will automatically import the files and compile the scripts. This may take a few minutes.

5. Open the project.

6. Open the "Map" scene.

7. Click the Play button at the top to run the game.

#### Download via Git

*Note: You must have Git and Unity version 6000.2.10f1 downloaded*
